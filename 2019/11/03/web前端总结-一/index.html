web前端总结（一）

1.什么是HTML?

HTML——超文本标记语言，以.html为扩展名。.htm网页文件

html语言–表名学习重点是代码部分。
标记–生成网页的代码都是单独出现或成对出现的。

浏览器

IE浏览器、FF火狐浏览器、Chrome谷歌浏览器、Safari苹果浏览器、O欧朋浏览器

不同的浏览器，显示的效果不同，通常和浏览器或设备的宽度有关。

***使用工具的方法：

  • 注意：站点名称不能是汉字，严格区分大小写，建议字母和数字组合。

四.基本的标签

1.使用标签的注意事项：

​ 无论何种标签，标签必须使用小写字母表示，属性必须使用小写字母，标点符号必须使用不在输入法情况下输入效果。标签必须正确的嵌套。
​ 使用单标签：注意/前加空格。

使用双标签：先写标签，再往标签内部插入其他标签或内容。

​ 网页中所有的内容都必须被标签包含，这样才能控制内容的修饰。加标签方便控制

2.为标签添加属性方法,在首标签中添加空格，激活软件的提示功能，进而进行选择属性的插入。

style属性：可以引出相关的属性对，一组属性对用表示结束。

align属性：可以控制水平对齐方式。

3.注释：针对代码进行解释，在网页中不显示。先写注释内容，然后鼠标拖拽选择注释内容，ctrl+/

4.自动跳转网页设置：

5.hn的标题：n表示1-6，1字体最大，6字体最小。外联元素。外联元素占满了一行。

6.p标签表示段落标记，外联元素。

***总结：所有的标签都分为不会自动换行的内联元素和会自动换行的外联元素，外联元素也叫块级 元素。

7.如何验证陌生标签是内联和外联元素？

外联元素会自动换行，可以添加对齐的属性，所以可以用来布局。

内联元素不会自动换行，不可以添加对齐的属性，无法布局，特点是只能进行修饰。
外联元素–h标题标签和p段落标签
内联元素–b加粗、u下划线、i倾斜、s删除、上标sup和下标sub。

8.相对路径和绝对路径

绝对路径–可以是根目录开头的路径，如d:/abc/1.jpg
–也可以是网上的网址路径，如http://www.baidu.com/img/1.jpg
相对路径–找文件之间的相对关系，平级文件直接写文件名。上一级文件用../表示。

9.a标签–内联元素超链接标记，自带字体和蓝色，有下划线效果，必须添加href属性，否则不会有 超链接的效果，如href没有具体的地址，用#表示空连接。target属性窗口的打开方式，_blank新 窗口打开，_self本窗口内跳转。
注意：a标签中的href和img标签中的src有区别的：
href是从当前文件连接过去。
src是把远处的文件拿过来展示。

10.图片标签img标签，src表示图片的路径，站点中是相对路径。alt图片的描述，title属性是图片的 提示信息，鼠标放在图片上的提示。切记，网页中图片不要用宽度width和高度height属性去控制 图片的尺寸，网页应用图片之前要事先修改好图片的大小。图片是内联元素。

11.图片的三种格式区别:

gif动图–也可以用来表示背景透明的图片，常用来表示颜色单一的logo图。

jpg图–网页常用的图片，放大后会失真，变模糊。
png图–对分辨率要求较高，文件占位比较大。

12.列表

  • ol有序列表，ul无序列表，dl自定义列表–列表都是外联元素。
  • ol有序列表–ol中属性可以用type设定有序列表的样式，1、a、A、i、I表示。用start表示从第几个列表符号开始。
  • ul列表中可以使用属性type，circle是空心点，disc实心点，square是矩形点。
  • dl表示自定义列表，dt表示列表名称，dd对于名称的解释

13、table表格-table表示表格，th表示列首（一列中第一个单元格），tr表示一行，td表示一个单元格。
table中的属性，border表格的边框，cellspacing表示表格的单元格之间的距离，cellpadding表示内容到单元格之间的内间距，bordercolor边框颜色，bgcolor表格的背景颜色，width宽度，height高度。
行占位符和列占位符的用法：操作合并单元格的方法
行占位符rowspan–占行成列的效果。
列占位符colspan–占列成行的效果。
切记一点：使用合并单元格，要先判断占行还是占列，然后添加占位符，删除对应的占位符。
14、在表格中的thead标签，只可以用对其方式控制水平位置的变化，而不能进行样式的修饰。
在表格中的tbody标签，使用方法遵循表格中tr和td内容显示，使用属性也都生效。
在表格中的tfoot表示页脚，页脚中也可以用tr和td去显示内容。可以使用对齐方式。
15、表单form标签，表单是搜集数据的，是外联元素，属性action表示将表单搜集的输入的内容传递到的目的地路径。
属性method表示，把搜集的数据用何种方式传递到action定义的地址中去。
在表单中，有三个主要的标签：input输入框 select选择下拉框 textarea文本域
注意：这三个标签都要被form标签去包含。
在input中最重要的是type属性，type属性表明是何种表单元素。

解释：type=”text”表明这是一个可以输入文本的输入框，value文本框中默认的内容，id给文本框加上id，maxlength表示这个文本框最大输入几个字符。因为html中所有内容要被包含，所以添加label标签包含用户名，label中属性for的值对应相关表单的id值，只能对应id，不能对应别的参数，id必须使用英文，不得使用汉字。

单选的type值是radio，为实现单选的效果，必须为input添加name属性。name值一致，才能出现单选的效果。

按钮也可以直接作为标签使用--如：<button>按钮</button>

16、表单中的下拉菜单select标签，包含的是option标签，option的属性有selected=”selected”选择默认的选项。
17、表单中的textarea表示文本域，rows表示文本域的高度，cols表示文本域的宽度。
