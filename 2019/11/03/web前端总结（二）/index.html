六.网页中的css样式

网页中的CSS样式：对于网页内容进行修饰。
CSS文件叫层叠样式表，用来修饰网页中内容的样式，层叠表示修饰样式是多层存在的。

  • CSS的文件扩展名是.css，在站点的css目录中建立层叠样式表的文件，一般用css.css或者style.css命名。
    CSS样式可以修饰的内容是：内联元素标签和外联元素的标签。

    内联元素的特征：不会换行，重点在修饰，用css修饰内联元素自身之外的属性。
外联元素的特征：自动换行，重点在布局，用css修饰外联元素进行布局时的宽度高度和位置等信息。

    总结：所有的HTML标签分为单标签和双标签，这两种标签都能用css修饰。

    所有的html标签分为内联元素和外联元素（块级元素），分别用css修饰各种的特点。

  • 使用css的方法，在css目录简历style.css的文件。然后再index.html入口文件的head部分，添加link的标签，link中的href属性来关联css的样式表文件。

rel属性表示样式的标签，tyle属性定义css的文本，href定义连接外部文件的路径。

测试是否连接成功：
在style.css中测试body{ background-color: blue;}，浏览器如果显示蓝色，表示连接成功。
其中body叫css的选择符，在{ }中的background-color表示body的背景颜色，blue设置body的背景颜色为蓝色。
注意：使用css时，第一行切记添加定义css的字符集。@charset “utf-8”;

  • 在网页文件index.html中的head标签中写入style的双标签，标签中写具体的css样式。

  • 在网页文件index.html中的head标签中写入style的双标签，标签中写具体的css的外部引用文件。

css的选择符：

  • 选择符{属性1：值1；属性2：值2；…}

  • 常用选择符：类型选择符、元素选择符、标签选择符

  • id选择符，给html的标签设置id属性，在css中，用#id名称进行引用。

注意：id选择符具有唯一性，所以id的名称只能定义一次。id并不是只能使用一次。

  • class选择符，类选择符，类选择符可以定义一次，可以多次使用。在css中用.加类名引用。

注意：class选择符和id选择符的区别：class可以定义多个，id只能定义一个。

  • 交集选择符，由类型选择符和id选择符或类型选择符和class选择符组合使用生成的选择符。

注意:交集选择符p.tiger{属性：值}一定类型选择符和类名引用相连，不能有空格。

  • 并集选择符–表示平级的选择符之间的关系。
  • 混合选择符，交集选择符，类型选择符，id选择符，class选择符，同时使用。
  • 包含选择符，也叫后代选择符—表示包含关系的选择符。
  • *通配符–表示所有的类型选择符都遵循此属性。 *{属性：值}

在CSS中的选择符的优先级：当选择符定义同一个标签的元素，鉴别以哪种选择符的属性为准。
id选择符>class选择符>类型选择符>通配符*
优先级中行内样式优先级最高！
注意：在选择符中，行内样式定义的CSS距离内容最近，所以优先级最高，也就是县官不如现管！
因为ID具有唯一性，而class可以定义多个，所以id大于class选择符。
class可以单独定义到具体的html标签，所以针对性比类型选择符强，所以优先级比类型选择高。

*代表全局，范围太广泛，所以优先级是最低的。

当我们使用样式，针对大多数的样式，可以使用* 进行定义，个别的样式单独存在的一种样式用id定义，凡是出现多次相同样式用class定义。
当用通配符进行定义了属性，其他选择符定义的属性和通配符相同，就会把*通配符定义的属性冲突掉。
